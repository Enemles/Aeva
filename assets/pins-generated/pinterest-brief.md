## ⚙️ Process de publication recommandé

1. **Étaler dans le temps** : 2-3 pins par jour pendant 4-5 jours (pas tout en une fois — Pinterest pénalise le spam)
2. **Heures optimales FR** : 18h-22h en semaine, 10h-13h le weekend (Pinterest analytics confirme après 1-2 semaines)
3. **Activer Rich Pins** : nécessite open graph propre sur les pages produit + catalog Shopify validé. Si pas encore validé, lance les pins quand même — Rich Pins se mettront à jour automatiquement quand le catalog passe.
4. **Ne pas raccourcir les liens** (bit.ly etc.) — Pinterest préfère les liens directs gopawly.pro
5. **Pas de hashtags dans le titre** (Pinterest les ignore en titre, ils comptent uniquement en description)

---

## 📊 Tracking après publication

Après 7 jours, vérifier dans Pinterest Analytics :
- Top pins en **impressions** (ce que l'algo pousse)
- Top pins en **clics sortants** (ce qui convertit)
- Top pins en **saves** (signal long terme — un save = ré-exposition gratuite plus tard)

Si un pin sort du lot en clics sortants, c'est ton angle de campagne ads quand tu dégèleras les 86€.
